# AI导诊助手

本项目是一个基于 **Vue 3**、**TypeScript** 和 **Arco Design** 的 Web 应用，旨在帮助患者通过大模型智能分析自身症状，辅助完成病情初判、科室推荐、在线挂号等一站式就医流程，提升就诊效率与体验。

---

##  技术栈

- **核心框架**：Vue 3（Composition API）
- **语言**：TypeScript
- **UI 组件库**：[Arco Design Vue](https://arco.design/vue)
- **构建工具**：Vite
- **包管理器**：pnpm（推荐）

---

##  快速启动

### 前置要求

- Node.js 版本 ≥ 18（推荐使用 [nvm](https://github.com/nvm-sh/nvm) 管理 Node 版本）
- 包管理器：推荐使用 **pnpm**（也可使用 npm 或 yarn）
- 浏览器：除了IE都行，IE不再支持vue3

> 如未安装 pnpm，可运行以下命令全局安装：
> ```bash
> npm install -g pnpm
> ```

### 启动步骤

1. 克隆项目：
   ```bash
   cd <你的目录>
    ```
   下面推荐使用SSH方式clone，请自行配置SSH密钥
   ```bash
   git clone git@github.com:Aaron-zjh-coder/AI_Medical_Tragile_Front.git
   ```
   cd <项目目录>
